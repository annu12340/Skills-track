Repo rescue mission

Modern guardrails for messy repos — without the corporate fluff.

Three bold skills. A clean story. One fast route from broken commits and legacy dependency chaos to code that ships with confidence.

Detective: Git Bisect AI

When a bug appears out of nowhere, this skill rewinds the repo and hunts the exact commit that brought the failure into the codebase.

  • Pinpoints the first bad commit with surgical precision.
  • Runs a tiny, reliable reproduction command instead of guessing.
  • Delivers an explanation, not just a commit hash.
Stop blaming the future. Find the culprit now.

Bouncer: Semantic Commit Guard

This one stands at the commit gate and checks the intent behind every change. It catches sketchy patterns before they ever become part of the repo.

  • Detects exposed secrets, architecture breaks, and missing docs.
  • Protects your repo from sloppy or risky commits.
  • Shows exactly what to fix when a commit is rejected.
Better than a late-night review rampage.

Exorcist: Dependency Resurrection Engine

Upgrade pain? This skill reads your dependency manifest, learns the breaking changes, rewrites code where needed, and runs the build again until the repo stands clean.

  • Parses `package.json` and `requirements.txt` with context.
  • Maps failures to real API and dependency fixes.
  • Retries the upgrade until the project builds successfully.
Keep the upgrade, ditch the dependency graveyard.
Why this matters: no more noisy bot prose, no more “it worked yesterday,” and no more build-breaking upgrades sitting in the dark. This page is the story of code that stays clean, commits that stay safe, and dependencies that stop haunting you.
